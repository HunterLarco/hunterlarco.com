From Scratch

Why? Why not use a framework? I build my personal projects from scratch because it has made me a better programmer. Rather than know how to use a relational database with integrated caching, I built one. Instead of using a ThreeJS, I build PT3: My own 3D JS Library. Each of these projects has exponentially increased my ability to tackle large projects, fully understand a system, and quickly acclimate to new systems. More importantly, when a framekwork fails, I can fix it because I do not rely on the frameworks I use. Why do I build from scratch? Because it allows me to push better products.
